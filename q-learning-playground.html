Q-Learning Algorithm

Playground soal untuk mahasiswa: isi parameter, matriks reward, matriks Q awal, dan hitung beberapa update Q-Learning berdasarkan lingkungan 6 state.

0/0 skor terakhir

Identitas

Soal

Robot berada pada lingkungan berisi state 0 sampai 5. State 5 adalah goal. Mahasiswa diminta menyusun komponen Q-Learning berdasarkan diagram, lalu menghitung nilai Q untuk episode latihan.

Gunakan nilai reward -1 untuk aksi yang tidak valid, 0 untuk perpindahan valid biasa, dan 100 untuk perpindahan yang mencapai goal.
Lingkungan state Goal: 5
goal valid action state awal acak

Langkah Algoritma

1

Set parameter gamma dan reward lingkungan pada matriks R.

2

Inisialisasi matriks Q dengan nilai nol.

3

Pilih initial state secara acak.

4

Jadikan initial state sebagai current state.

5

Pilih satu aksi yang mungkin dari current state.

6

Gunakan aksi tersebut untuk menuju next state.

7

Ambil nilai Q maksimum dari next state untuk semua aksi yang mungkin.

8

Hitung Q(state, action) = R(state, action) + gamma * max Q(next state, semua aksi).

9

Ulangi proses sampai current state sama dengan goal state.

Bagian A: Parameter

Isi gamma sesuai instruksi dosen untuk latihan ini.

Bagian B: Matriks Reward R

Isi matriks R berdasarkan diagram lingkungan. Baris adalah current state, kolom adalah action atau next state.

Bagian C: Matriks Q Awal

Isi matriks Q awal sebelum proses pembelajaran dimulai.

Bagian D: Hitung Update Q

Gunakan gamma = 0.8 dan Q awal bernilai nol. Hitung berurutan dari baris 1 sampai 5, sehingga hasil pada baris sebelumnya boleh memengaruhi max Q pada baris berikutnya.

Bagian E: Cara Hitung Tiap Langkah

Lengkapi perhitungan setiap langkah. Gunakan rumus Q(state, action) = R(state, action) + gamma * max Q(next state), dengan gamma = 0.8.

Bagian F: Matriks Q Hasil Update

Isi matriks Q awal dan matriks Q setelah setiap langkah update. Sel kuning menunjukkan posisi Q yang berubah pada langkah tersebut.
